Frontend Development

The front end of a website is the part that users can see and interact with. This includes the graphical user interface (GUI), command line, design, menus, texts, images, and videos.


For you to be a front-end developer you have to learn various technologies. Here are some of them.

  • HTML
    This is markup language giving structure to the website
  • CSS
    This is cascading style Sheet used to beautify a website
  • Javascript
    This is a programming language that provides functionality to the website

You can also learn these technologies, they can come in handy.

  • Bootstrap
  • React
  • Vuejs
  • Angular
  • jQuery
